Visualization done using DataPane Library. All countries along with it's date wise price can be seen while hovering the graph and countries can be set using dropdown menu provided.

Time Series Analysis done using ARIMA Model.
I have verified the data for 100 days of the gold prices and got the results as follows.
Test RMSE: 1472.953
Test MAE: 1049.322
Test MAPE: 0.008

I have also predicted the gold prices for the next 20 days from the last date of the dataset.
